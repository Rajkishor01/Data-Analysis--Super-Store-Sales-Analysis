# Data-Analysis : Super-Store-Sales-Analysis

## Project Overview : 
This project analyzes retail sales data to evaluate sales performance, profitability, customer behavior, and shipping efficiency using Power BI and Power Query.

## Dataset Overview :
  - Rows= 51295
  - Sheets : Original- Order, Retunrs, Person
           : Derived- Merge11, Simple Analysis Report
## Data Cleaning and Transformation :
   - Merged sheets
   - Changed data types
   - Removed duplicates
   - Replaced null values
   - Pivot tables of counts of Important categorical values
## Power BI DashBoard :
   - Sales and Profit Analysis
   - Customer and Product Analysis
   - Shipping and Regional analysis
   - { Created new column: Shipping Duration in Power BI}
## Key Insights :
  - UNITED STATES have most profit
  - Average Shipping Days : 4
  - CENTRAL region has most profit
